# ert-fmcw-radar
Handheld geophysical sensor fusing Electrical Resistivity Tomography (ERT) and 24 GHz FMCW Radar for subsurface imaging. Arduino firmware reads a 16-bit ADS1115 ADC + LD2410C radar module, applies baseline calibration and EMA smoothing, and streams fused scan data. Built for paleontology/archaeology fieldwork.
